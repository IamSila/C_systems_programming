# file system
- Everything is a file in linux.

# File metadata inodes
- `inode` is a data structure that tracks all information about a file.
- It has the following structure:
    ```
    size
    mode
    owner information
    permision
    Location
    UID
    GID
    Timestamp
    Access control
    Direct block
    Indirect block
    Double Indirect block
    Tripple indirect block
    ```
